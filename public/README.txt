Drop your 1-hour music track here as:  music.mp3

(mp3, ogg, or wav all work — but the file MUST be named exactly "music.mp3"
for it to auto-load. Alternatively, use the file picker on the start screen
to load any audio file on the fly.)

If no music.mp3 is present, the app plays a built-in generative ambient pad
so the world still reacts to sound.
